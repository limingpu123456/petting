基于springboot+vue的宠物交易/领养系统 springboot+mybatis+mysql+vue

一、功能介绍
包括管理员、普通用户两种角色，外加游客(未登录情况)

权限  游客 < 普通用户 < 管理员

1、游客功能

查看宠物信息

2、普通用户功能

出售/领养宠物、购买/收养宠物、订单管理、地址管理

和卖家/卖家对话

个人信息修改、密码修改

3、管理员功能

用户管理、地址管理、宠物管理、订单管理

个人信息、密码修改
